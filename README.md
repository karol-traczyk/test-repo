> **Did you know?** The iconic cat that Marlon Brando holds in the opening scene of *The Godfather* was not part of the original script. Director Francis Ford Coppola found the stray cat wandering around the studio lot and placed it in Brando's hands just before filming. The cat purred so loudly during the scene that it muffled some of Brando's dialogue, requiring parts of it to be looped in post-production!
